# Barber Booking App

A full-stack mobile application for booking barber services.

## Tech Stack
- **Frontend**: React Native (Expo), TypeScript
- **Backend**: Node.js, Express, TypeScript, MySQL

## Prerequisites
- Node.js
- XAMPP (for MySQL)
- Expo Go app on your phone (or Android Emulator/iOS Simulator)

## Setup

### Backend
1.  Navigate to `backend` directory.
2.  Run `npm install`.
3.  Create `.env` file (see `env.example` or code).
4.  Run `npm run start` (this will also initialize the database if it doesn't exist).

### Frontend
1.  Navigate to `frontend` directory.
2.  Run `npm install`.
3.  Run `npx expo start`.

## Features
- User Registration & Login
- Browse Services
- View Service Details
- Book Appointments
- View My Bookings
